# fldanplay

fldanplay 是一款跨平台的本地/网络媒体播放应用，支持弹幕显示。

#### ⚠️ 本项目处于开发阶段，不保证稳定性和数据安全。
---

## 支持平台
- Android
- Windows (beta)
- Linux

## 计划支持平台
- macOS (缺少设备适配)
- iOS (缺少设备适配)
- Android TV

## 快速上手
1) 安装并启动应用。
2) 首次进入推荐在设置中进行配置（Android推荐下载字体作为字幕字体）。
3) 在设置中配置弹幕服务。
4) 在首页右下角添加媒体库。
5) enjoy!
